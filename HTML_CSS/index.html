Tools and Technologies to Learn MERN Stack

    • Git Commands

    1. git init → To create a repository.
    2. git add → It adds files to the staging area.
    3. git commit -m “message” → It commits the files in the staging area with a message.
    4. git push -u origin main → It pushes the files from staging area to the github repository.
    5. git status → Shows the status of the repository.
    6. git remote add origin “repo link”→ To connect the remote repo to local repo.
    7. git pull → It pulls the changes from the remote repo to the local repo
    8. git log → Shows the log of the repository.
    9. git config → To config the username and email of the repository.
